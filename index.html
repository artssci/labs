What is Ascii Art

ASCII art is a graphic design technique that uses computers for presentation and consists of pictures pieced together from characters defined by the ASCII Standard from 1963. The term is also loosely used to refer to text-based visual art in general. ASCII art can be created with any text editor, and is often used with free-form languages

Creating your own Ascii Art

Now that you have experimented with HTML and Glitch.com, follow the next steps to create your own Ascii Art.

  1. Go to the following Ascii generator page: https://www.ascii-art-generator.org/
  2. Follow the instructions on how to generate Ascii Art from a picture or text
  3. Once it has been generated, click Download result and save it in the computer's Desktop
  4. Open the downloaded file and copy the generated Ascii Art
  5. Go to your HTML code in Glitch and add the tags <pre></pre> below the <p></p> tags
  6. Paste the Ascii Art between <pre></pre>. The <pre> tag allows the format of the text to stay unchanged. See the example below:
    <pre>
