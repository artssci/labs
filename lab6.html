What is CSS grid layout

CSS Grid Layout provides a two dimensional layout system, controlling layout in rows and columns. When creating a grid layout you define a grid with rows and columns. Then you place items onto that grid, or allow the browser to auto-place them into the cells you have created. ... click to hear a 30-min podcast about CSS Grid!

Define a grid layout with CSS

  1. Remix the following Glitch webpage that contains some basic code to get you started: https://glitch.com/edit/#!/web-grid-layout-lab
  2. Go to the style.css file and start defining a grid layout by typing the code display: grid; inside the class selector named .grid-layout
  3. Then, set the number of columns of the grid by typing the code grid-template-columns: repeat(5, 1fr); still inside the .grid-layout selector

    4. The grid-template-columns property defines the number of columns in your grid layout, and it can define the width of each column.

  4. Now, set the number of rows of the grid by typing the code grid-template-rows: repeat(5, 1fr); still inside the .grid-layout selector

    5. The grid-template-rows property defines the number of rows in your grid layout as well as the height of each row.

  5. Check the example code and the resulting image below

Unlike Illustrator, in CSS it is not possible to see/display the lines of the grid layout. That is why you have to use the wireframe from Illustrator to guide yourself 

.grid-layout {
  width: 100vw;
  height: 100vh; 
  display: grid;
  grid-template-columns: repeat(5, 1fr);
  grid-template-rows: repeat(5, 1fr);
}

Place each section of your high-fidelity wireframe into the CSS grid

Similarly to Illustrator, in CSS Grid layouts, the lines between columns are called column lines. The lines between rows are called row lines. Refer to line numbers when placing a grid item in a grid layout.

  1. Still in the style.css file define in which column line each section will start. For example, if your 1st section starts in the 1st column line then type grid-column-start:1;
  2. Then, define in which column line each section will end. For example, if your 1st section ends in the 6th column line then type grid-column-end:6;
  3. Now, define in which row line each section will start. For example, if your 1st section start in the 1st row line then type grid-row-start: 1;
  4. Finally, define in which row line each section will end. For example, if your 1st section ends in the 2nd row line then type grid-row-end: 2;
  5. Check the example code and the resulting image below
.section-one {
  background: #808080;
  grid-column-start: 1;
  grid-column-end: 6;
  grid-row-start: 1;
  grid-row-end: 2;
}

.section-two {
  background: #989898;
  grid-column-start: 1;
  grid-column-end: 3;
  grid-row-start: 2;
  grid-row-end: 5;
}


.section-three {
  background: #C0C0C0;
  grid-column-start: 3;
  grid-column-end: 6;
  grid-row-start: 2;
  grid-row-end: 5;
}

.section-four {
  background: #808080;
  grid-column-start: 1;
  grid-column-end: 6;
  grid-row-start: 5;
  grid-row-end: 6;
}

Add text, adjust the text color, and select font families accordingly

  1. In the index.html file use the <h1> </h1> tags to add headings and subheadings as you see it fits, based on your high-fidelity wireframe

    2. HTML headings are defined with the <h1> to <h6> tags. <h1> defines the most important heading. <h6> defines the least important heading. More info here: https://www.w3schools.com/html/html_headings.asp

  2. Use the <p> </p> tags to add paragraphs of text as you see it fits
  3. In the style.css file change the text color using the color property we saw last week. For example color:green; or color:#038C73
  4. Select a font family similar to the one in your wireframe from this page: https://www.w3schools.com/cssref/css_websafe_fonts.php Use the font-family property. For example: font-family: Arial, sans-serif;

    5. You might not be able to find the exact font family in the link above. For now, choose one that looks similar and I will show you how to access more font families next week.

  5. See the code example and image below
.section-one {

  color: white;
  font-family: Garamond, serif;
}

   <img src="image URL goes here" alt="image description">
  </section>
