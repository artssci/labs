Export your illustration in SVG format and with ID attributes

Check the following interactive tutorial to learn how to export your illustration in SVG format and with ID attributes

7 STEPS

1. Click the main Frame to select the whole illustration

2. Scroll down to Export and select the SVG format

3. Click the 3 dots to see more options

4. Select Include "id" attribute

5. Close the window

6. Click Export Magic Potion to save the SVG in the computer

7. That's it. You're done.
