What is Creative Coding

Creative coding is a type of computer programming in which the goal is to create something expressive instead of something functional. It is used to create live visuals and for VJing, as well as creating visual art and design, entertainment (e.g. video games), art installations, projections and projection mapping, sound art, advertising, product prototypes, and much more. ... click to read the full article

 function preload() {

5.2 Use the cork, bottle, and potion Id selectors to trigger the sounds

  1. Go to the sketch.js file
  2. Under the function setup(), type the Id names of the cork, bottle, and potion you identified in step 5.1 inside the select() function. Include a hashtag (#) before each name
  3. See the example code below:
function setup() {
  noCanvas();
  select("#cork").mousePressed(playCork);
  select("#potion").mousePressed(playPotion);
  select("#bottle").mousePressed(playBottle);
}

The functions select() and mousePressed() will trigger the functions playCork(), playPotion(), and playBottle() which contain the function play() which plays the sounds you defined in the variables at the top of of the file, under the function preload()
