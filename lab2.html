How Figma relates to Scalable Vector Graphics

  • When you create designs in Figma, you are essentially working with vector elements that can be exported as SVG files.
  • In Figma, when you drag a rectangle or adjust its properties, you are actually creating and modifying SVG code behind the scenes. For example, when you:
    • Draw a rectangle, Figma creates a <rect> element
    • Round its corners, Figma adds rx and ry attributes
    • Change its color, Figma modifies the fill attribute
  • Figma is a "What You See Is What You Get" (WYSIWYG) interface where you manipulate visual elements directly and see exactly how the final result will look while you are working.

Explore Figma's basic shapes

  1. Open Figma and select Create New > Design File
  2. Name your file: Use the navigation panel (left panel) to name your file "lab2 - basic shapes"
  3. Create a frame: Use the Frame Tool in the toolbar to draw a frame with dimensions 1100 x 500.
  4. Draw shapes: Inside the frame, use the Shape Tools to draw a square, a circle, an ellipse, a line, a triangle, and a pentagon
  5. Refer to this tutorial for guidance: https://help.figma.com/hc/en-us/articles/360040450133-Shape-tools

Export your Figma file as SVG and explore it in VS Code

Follow the interactive tutorial below to learn how to export your Figma file as an SVG.

6 STEPS

1. Click the frame that contains all the shapes

2. Locate the Export option at the bottom of the properties panel (right sidebar)

3. Click to see the format options

4. Select the SVG format

5. Click Export and save the file in your computer

6. That's it. You're done.
