How Figma relates to Scalable Vector Graphics

  • When you create designs in Figma, you are essentially working with vector elements that can be exported as SVG files.
  • Figma is "What You See Is What You Get" (WYSIWYG) interface where you manipulate visual elements directly and see exactly how the final result will look while you are working.
  • In Figma, when you drag a rectangle or adjust its properties, you're actually creating and modifying SVG code behind the scenes. For example, when you:
    • Draw a rectangle → Creates a <rect> element
    • Round its corners → Adds rx and ry attributes
    • Change its color → Modifies the fill attribute

  4. See the changes by opening your file with a web browser or an image viewer

Circle 


<circle r="50" cx="250" cy="100" />
  1. circleis the SVG element that creates a circle.
  2. r="50" sets the radius to 50 units - this determines how big the circle is.
  3. cx="250" positions the center of the circle 250 units from the left edge.
  4. cy="100" positions the center of the circle 100 units from the top edge.

Ellipse 


<ellipse rx="100" ry="50" cx="450" cy="100" />
  1. ellipseis the SVG element that creates an oval shape.
  2. rx="100" sets the horizontal radius to 100 units - this determines how wide the ellipse is.
  3. ry="50" sets the vertical radius to 50 units - this determines how tall the ellipse is.
  4. cx="450" positions the center of the ellipse 450 units from the left edge.
  5. cy="100" positions the center of the ellipse 100 units from the top edge.

Line 


<line x1="600" y1="100" x2="750" y2="100" stroke="black" />
  1. line creates a straight line between two points
  2. x1="600" sets the starting point 600 units from the left edge
  3. y1="100" sets the starting point 100 units from the top edge
  4. x2="750" sets the ending point 750 units from the left edge
  5. y2="100" sets the ending point 100 units from the top edge
  6. stroke="black" sets the line color to black

Polygon 


<polygon points="850,50 900,150 800,150" />
  1. polygon creates a closed shape made up of straight lines between points
  2. points="850,50 900,150 800,150" defines the coordinates for each corner:
    • First point: (850,50) - 850 units from left, 50 units from top
    • Second point: (900,150) - 900 units from left, 150 units from top
    • Third point: (800,150) - 800 units from left, 150 units from top

Path 


<path d="M1000 50 L1050 100 L1025 150 L975 150 L950 100 Z" />
  1. path is the most flexible SVG element - it can create any shape using commands
  2. The d attribute contains the drawing commands
    • M1000 50 means "Move to" position (1000,50) without drawing
    • L1050 100 means "Line to" position (1050,100)
    • L1025 150 means "Line to" position (1025,150)
    • L975 100 means "Line to" position (975,100)
    • Z means "Close path" - draws a line back to starting point

Rectangle 


<rect 
width="100" 
height="100" 
x="50" 
y="50" 
transform="translate(0 200)" 
fill="blue"  />
  1. transform="translate(0 200)" moves (translates) the rectangle from its original position
    • The values (0, 200) mean:
      • 0 units horizontally (no left/right movement)
      • 200 units down vertically
      • This translation happens AFTER the initial positioning (x="50" y="50")
      • So the final position would be:
        • 50 units from the left (from x="50")
        • 250 units from the top (50 from y="50" plus 200 from translate)
  2. fill="blue" changes the color from the default black to blue

Polygon 


<polygon 
points="850,50 900,150 800,150" 
transform="translate(0 200)" 
style="fill:lime;stroke:purple;stroke-width:3"/>
  1. transform="translate(0 200)" moves (translates) the polygon down 200 units from its original position
    • So all points shift down 200 units from their original position
    • The top point is now at (850,250)
  2. fill:lime sets the shape color to lime green.
  3. stroke:purple sets the line color to purple.
  4. stroke-width="3" sets the line width to 3 pixels/units
