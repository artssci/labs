User interaction: Create a navigation bar with HTML (and CSS)

A navigation bar is a link to appropriate sections/pages in a website that helps readers in traversing the online document. Considered a traditional method of navigation, a navigation bar can be implemented in a number of ways, namely, horizontally or vertically, or fixed or dynamic. A navigation bar implementation is considered one of the key points of Web design and usability. ... click to read the full article

Add a high color contrast mode with HTML, CSS, and JavaScript

You may have found that sometimes, websites have a high-contrast mode that allows users to read the text better by switching from a regular color palette to a high-contrast one, such as a black-and-white one. Follow the instructions below to add a high-contrast mode using either a black background and white text or a white background and black text:

4.1 Add buttons for the user to switch from the initial color palette to a high-contrast color palette

  1. Go to the index.html file and add a button element using the <button> </button> tags. Add it to the first section
  2. Then, add the onclick= attribute along with a name made by you, such as "highContrastCSS()". Make sure that the name is between quotation marks and ends with a parenthesis
  3. Now, choose a title for the button and type it between the button tags. For mine, I chose High Contrast Mode. See example:

    4. <button onclick="highContrastPaletteCSS()">High-Contrast Mode</button>

  4. Finally, repeat the same process for another button, which will allow the user to switch back to the initial color palette with lower contrast
  5. Check the example code and the resulting image below:
<section class="section-one"> 
  <h2>MEDIAART 2A06 - DESIGN AND CODE</h2>
  <button onclick="highContrastPaletteCSS()">High Contrast Mode</button>
  <button onclick="initialPaletteCSS()">Initial Contrast Mode</button>
</section>

4.2 Use JavaScript to program the High Contrast Mode button

At this point, you might have noticed that by clicking the buttons you coded above, nothing changes. This is because you need to program these changes with JavaScript. Follow the steps below to learn how to program them:

  1. Create a new JavaScript file by clicking on Files in your Glitch project. Then, give it a name such as script.js and press enter
  2. Go to the newly created script.js file, which should be found listed along with the index.html and the style.css files
  3. Create a new function using the function command followed by the name of the 1st button you created in the section above and two curly brackets. See example:
    4. function highContrastPaletteCSS(){}
  4. Inside the curly brackets, type the following line of code: document.getElementsByClassName("section-one")[0].style.background = "black";
  5. Now, click the High Contrast Mode button and see how the background color of the 1st section changes to black
  6. Repeat step 4 to change the color of other sections to black. For each line of code you add, you will need to change the class name from "section-one" to "section-two" and so on
  7. See example code and image below:
function highContrastPaletteCSS(){
  document.getElementsByClassName("section-one")[0].style.background = "black";
  document.getElementsByClassName("navigation-bar")[0].style.background = "black";
  document.getElementsByClassName("section-two")[0].style.background = "black";
  document.getElementsByClassName("section-three")[0].style.background = "black";
}

4.3 Use JavaScript to program the Initial Contrast Mode button

Now, let's program the Initial Contrast Mode Button for the user to be able to switch back to the Initial Contrast Mode. Follow the steps below:

  1. Still in the script.js file, create another function using the function command followed by the name of the initial contrast button and two curly brackets. See example:
    2. function initialPaletteCSS(){}
  2. Inside the curly brackets, type the following line of code: document.getElementsByClassName("section-one")[0].style.background = "#8C1B24";
  3. Now, click the Initial Contrast Mode button and see how the background color of the 1st section changes back to the initial color (i.e., #8C1B24)
  4. Repeat step 2 to change the color of other sections to their initial color and contrast. You can find the initial color of each section by looking at the style.css file. For each line of code you add, you will need to change the class name from "section-one" to "section-two" and so on
  5. See example code and image below:
function initialPaletteCSS(){
  document.getElementsByClassName("section-one")[0].style.background = "#8C1B24";
  document.getElementsByClassName("navigation-bar")[0].style.background = "#F25252";
  document.getElementsByClassName("section-two")[0].style.background = "#162040";
  document.getElementsByClassName("section-three")[0].style.background = "#8C1B24";
}

Add a font resizing option to make the text more legible

The Web Content Accessibility Guidelines (WCAG), an organization that defines standards for web content accessibility, does not specify a minimum font size for the web. But we know there’s such a thing as text that is too small to be legible, just as text that can be too large to consume. So, how can we make sure our font sizes are accessible? What sort of best practices can we rely on to make for an accessible reading experience?

We want to provide users with low vision a way to choose how fonts are displayed. Not in a crazy way. More like the ability to increase the size by 200% while maintaining readability and avoiding content collisions and overlaps ... click to read the full article

5.1 Add buttons for the user to switch from the initial font size to a larger font size

  1. Go to the index.html file and add a button element, using the <button> </button> tags. Add it to the first section, below the buttons you added in section 4
  2. Then, add the onclick= attribute along with a name made by you, such as "fontSizeLarge()". Make sure that the name is between quotation marks and ends with a parenthesis
  3. Now, choose a title for the button and type it between the button tags.For mine, I chose Large Font-Size. See example:
    4. <button onclick="fontSizeLarge()"">Large Font-Size</button>

  4. Finally, repeat the steps 2 and 3 to add a second button that will allow the user to switch back to the initial font size
  5. Check the example code and the resulting image below
<section class="section-one"> 
  <h2>MEDIAART 2A06 - DESIGN AND CODE</h2>
  <button onclick="highContrastPaletteCSS()">High Contrast Mode</button>
  <button onclick="initialPaletteCSS()">Initial Contrast Mode</button<
  <button onclick="fontSizeLarge()">Large Font Size</button>
  <button onclick="fontSizeInitial()">Initial Font Size</button<
</section>

6.2 Use JavaScript to program the Large Font Size button

Now, let's program the Large Font Size button. Follow the steps below:

  1. In the script.js file, create a function with the name of the large font size button. See example: function fontSizeLarge(){}
  2. Inside the curly brackets, type the following line of code: document.getElementsByClassName("section-one")[0].style.fontSize = "x-large";
  3. Now, click the Large Font Size button and see how the font size of the 1st section changes to extra large
  4. Repeat step 2 to change the font size of other sections to extra large font
  5. See example code and image below:
function fontSizeLarge(){
  document.getElementsByClassName("section-one")[0].style.fontSize = "x-large";
  document.getElementsByClassName("navigation-bar")[0].style.fontSize = "x-large";
  document.getElementsByClassName("section-two")[0].style.fontSize = "x-large";
  document.getElementsByClassName("section-three")[0].style.fontSize = "x-large";
}

6.3 Use JavaScript to program the Initial Font Size button

Finally, let's program the Initial Font Size button for the user to switch back to the initial font size. Follow the steps below:

  1. In the script.js file, create a function with the name of the initial font size button. See example: function fontSizeInitial(){}
  2. Inside the curly brackets, type the following line of code: document.getElementsByClassName("section-one")[0].style.fontSize = "initial";
  3. Now, click the Initial Font Size button and see how the font of section 1 resizes to its initial size
  4. Repeat step 2 to change the font size of other sections to their initial size
  5. See the example code and image below:
function fontSizeInitial(){
  document.getElementsByClassName("section-one")[0].style.fontSize = "initial";
  document.getElementsByClassName("navigation-bar")[0].style.fontSize = "initial";
  document.getElementsByClassName("section-two")[0].style.fontSize = "initial";
  document.getElementsByClassName("section-three")[0].style.fontSize = "initial";
}
