What is Vector Art and the SVG format

Vector artwork is a term that describes any art made with vector illustration software like Adobe Illustrator. Vector artwork is built from vector graphics, which are images created with mathematical formulas.

The SVG (Scalable Vector Graphics) file format is a popular tool for displaying two-dimensional graphics, charts, and illustrations on websites. Complete 1 of the artworks and create 2 variations

Next Thursday, you will need some of these artworks to do the lab exercise related to generative art. Complete 1 of the artworks and create 2 variations for Thursday's lab. Follow the steps below:

  1. Choose any of the artwork you worked on in steps 2, 3, or 4
  2. Complete 1 of the artworks you worked on (e.g., the monster, the car, the doughnut in a trailer, etc)
  3. Then, create 2 additional new versions of your chosen art in Illustrator. Each variation should have different characteristics. For example, they could either have different colors, sizes, or orientations.
  4. You can create these new variations on the same artboard or in a new artboard. See how to create a new artboard here: https://helpx.adobe.com/ca/illustrator/using/introduction-to-artboards.html

Export the completed artwork and its 2 variations to SVG format

Export each artwork individually so you get 3 SVG files (i.e., 1 per artwork). This way, you will be able to use them individually next Thursday during the Generative Art exercise

  1. Once you have the completed artwork and its 2 variations, in Illustrator, click File -> Export -> Export As
  2. Then, select a folder to save the completed original artwork plus the 2 additional versions, give them a name, and click save as SVG
  3. Copy them to a USB key or send them to yourself through email to have them ready next Thursday
