What is Vector Art

Vector artwork is a term that describes any art made with vector illustration software like Adobe Illustrator. Vector artwork is built from vector graphics, which are images created with mathematical formulas. In comparison, raster art (also referred to as bitmaps or raster images) is created with colorized pixels. Enlarge pixel-based art in a raster file too much and it looks jaggy, whereas you can enlarge vector art to any size without negatively affecting its appearance. This resolution independence allows vector art to be used in a variety of forms, from small illustrations to massive billboards. ... click to read the full article

Incorporate your Vector Art into your Sketchbook

Follow the steps below to incorporate your Vector Art into the Sketchbook webpage you worked on last Tuesday

7.1 Select and export your Vector Art as an SVG

The SVG file format is a popular tool for displaying two-dimensional graphics, charts, and illustrations on websites. Plus, as a vector file, it can be scaled up or down without losing any of its resolution ... click to read the full article

  1. Choose any of the artwork you did in steps 3, 4 or 5
  2. In Illustrator, click File -> Export -> Export As
  3. Then, select a folder to save it, give it a name, and click save as SVG

Watch the following video to learn how to export individual icons/objects in Illustrator. Don't forget that you need to export it as SVG as opposed to the PNG format mentioned in the video:


7.2 Display your Vector artwork in your Sketchbook cover

  1. Upload the exported vector art to your Glitch project from last Tuesday (i.e., the Sketchbook
  2. In the index.html file substitute the current <img> link under the second section with the link to your SVG vector art
  3. Preview the webpage to see your vector art displayed instead of the vector art tree that was previously displayed
  4. Resize it accordingly by going to the style.css file and adding the img{} selector with the width: attribute and a value in percentage such as 30%;

    If you are in doubt about how to resize an image, please refer back to lab #4, step 6, where we displayed and resized images: https://mediaart2a06.glitch.me/lab4.html
