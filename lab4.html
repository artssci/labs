Image display options with CSS

To finalize the display of your photos, explore different display direction options. Do the following:
  1. In the style.css file type the CSS class selector of the section where your images are contained. Inside the class selector type the display property.
  2. Set the value of the display property to flex
  3. Below the display property add the flex-direction property
  4. Set the value of the flex-direction property to row
  5. Preview your webpage to see how the photos are displayed in a row. Try other flex-direction values such as row-reverse;, column;, and column-reverse;
Try the steps above in the interactive tutorial below:
