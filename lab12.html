What is Generative Art

Generative art refers to art that in whole or in part has been created with the use of an autonomous system. An autonomous system in this context is generally one that is non-human and can independently determine features of an artwork that would otherwise require decisions made directly by the artist.

"Generative art" often refers to algorithmic art (algorithmically determined computer generated artwork) and synthetic media (general term for any algorithmically generated media), but artists can also make it using systems of chemistry, biology, mechanics and robotics, smart materials, manual randomization, mathematics, data mapping, symmetry, tiling, and more ... click to read the full article

Create a row of your SVG illustrations

Let's make your Generative Art piece more impactful by creating a row with multiple copies of your SVG illustrations. To do this, you will use a for loop. Continue to the next section to know what is a for loop.

4.1 Use a for loop to create multiple copies of your SVG illustrations displaying horizontally

A "for loop" creates a loop that is useful for executing one section of code multiple times ... click to read the full article

  1. Add a for loop in the sketch.js file, just in between the myRandomSVG= variable and the function image(). Check the following code:
    for (var x = 0; x <= width; x = x+100)

    2. A 'for loop' consists of three different expressions inside of a parenthesis to control the number of times the loop is run:
    - The 1s expression is used to set the initial state for the loop
    - The 2nd expression is a condition that you would like the computer to check before each loop. If this expression returns false, then the loop will exit
    - The 3rd expression is executed at the end of each loop

  2. Just after the for loop, type an opening curly bracket, {. Then, type a closing curly bracket, }, after the image() function. This way, the image() function will be the section of code that will be executed multiple times by the for loop
  3. Check the example code: 
for (var x = 0; x <= width; x = x+100) {
  image(myRandomSVG, 0, 0, 100, 100);
}

At this point, you won't be able to see the illustration copies because they are all stacked together. Continue to the next section to move each illustration to a different position horizontally

Create columns for each row of SVG illustrations

Now, let's fill the rest of the webpage with copies of your illustrations positioned vertically in columns. You will use another for loop.

6.1 Use a for loop to create multiple copies of your SVG illustrations vertically

Add a for loop inside of the previous for loop. Follow the instructions below to learn how:

  1. Add another for loop below the 1st for loop. Change the variable name x to y. Also, change the variable width to height. Check the following code: for (var y = 0; y <= height; y = y+100)

    2. Width and height are "system variables" that store the width and height of the drawing canvas or the screen in the case of our code

  2. Just after the 2nd for loop, type an opening curly bracket, {. Then, type a closing curly bracket, }, after the image() function. This way, the 2nd for loop, along with the image() function, will be executed multiple times by the 1st for loop
  3. Check the example code: 
for (var x = 0; x <= width; x = x+100) {
  for (var y = 0; y <= height; y = y+100) {
   image(myRandomSVG, x, 0, 100, 100);
 }
}

You won't be able to see the vertical illustration copies because they are all stacked together. Continue to the next section to move each illustration to a different position vertically
